# PWA Icons

Letakkan ikon PWA di folder ini:

- `icon-192.png` (192×192)
- `icon-512.png` (512×512)

Saat ini manifest mengacu ke kedua file tersebut.

Jika kamu belum punya asset final, buat placeholder sederhana (mis. dari Figma) dengan background `#0f172a`
dan teks/logo “SIS” agar Lighthouse PWA tidak mengurangi skor karena ikon tidak lengkap.
